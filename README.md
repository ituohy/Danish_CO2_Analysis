# Denmark CO₂ Emissions Analysis

An exploratory data analysis of Denmark's CO₂ emissions from 1950 to present,
built with Python as part of my data analysis portfolio.

## 📊 Key Questions
- How have Denmark's total CO₂ emissions changed over time?
- How has emissions per capita shifted?
- Which energy sources are driving emissions?

## 🔍 Key Findings
- Emissions peaked in the mid-1990s and have declined sharply since
- CO₂ per capita has fallen from ~12 to under 5 tonnes per person
- Coal has declined significantly; oil remains the dominant source
- The post-1990 decline aligns with Denmark's Kyoto Protocol commitments

## 🛠️ Tools Used
- Python, Pandas, Matplotlib, Seaborn
- Data source: [Our World in Data CO₂ Dataset](https://github.com/owid/co2-data)

## 📁 Files
- `analysis.ipynb` — full Jupyter notebook with code and commentary
- `co2_total.png` — total emissions over time
- `co2_per_capita.png` — per capita emissions over time
- `co2_by_source.png` — emissions broken down by source
